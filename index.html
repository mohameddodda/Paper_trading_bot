iPhone / iPad (Pythonista 3)

  1. Install Pythonista 3 from App Store
  2. Tap + → Import from GitHub → MohamedDodda/Paper_trading_bot
  3. Run pythonista_ios_mode.py → type start

PC / Mac / Linux (Recommended)

  1. git clone https://github.com/MohamedDodda/Paper_trading_bot.git
  2. pip install -r requirements.txt
  3. set OPENROUTER_API_KEY=sk-... (Windows) or export (Mac/Linux)
  4. python bot.py

Your API key stays private on your device.
